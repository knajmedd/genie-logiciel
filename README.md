# Génie Logiciel

Projet de génie logiciel (travaux pratiques et livrables). Ce dépôt centralise la documentation et les livrables associés.


## Prérequis
- Windows avec WSL (Ubuntu 22.04).
- Git installé et configuré (nom, email).
- VS Code.



## Structure du dépôt
```text
genie-logiciel/
├── README.md
├── delivrables/
│   └── note.md
└── 2048-js/
	├── README.md
	├── index.html
	├── style.css
	└── script.js
```

## Utilisation
- Consulter les documents dans `delivrables/`.
- Mettre à jour `note.md` au fil des séances.
- Documenter tout changement significatif dans ce README.

### Lancer le projet 2048 (HTML/CSS/JS)
Dans `2048-js/`:

```bash
# Ouvrir directement index.html dans le navigateur

# ou lancer un petit serveur local
cd 2048-js
python -m http.server 8000
# puis visiter http://localhost:8000
```
