# Génie Logiciel

Projet de génie logiciel (travaux pratiques et livrables). Ce dépôt centralise la documentation et les livrables associés.


## Prérequis
- Windows avec WSL (Ubuntu 22.04).
- Git installé et configuré (nom, email).
- VS Code.



## Structure du dépôt
```text
genie-logiciel/
├── README.md
└── delivrables/
	└── note.md
```

## Utilisation
- Consulter les documents dans `delivrables/`.
- Mettre à jour `note.md` au fil des séances.
- Documenter tout changement significatif dans ce README.
